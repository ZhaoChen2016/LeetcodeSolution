#双指针
##前后双指针
19 160
即一个指针在链表中提前朝着指向下一个节点的指针移动若干步，然后移动第2个指针。前后双指针的经典应用是查找链表的倒数第k个节点。
##快慢双指针
两个指针在链表中移动的速度不一样，通常是快的指针朝着指向下一个节点的指针一次移动两步，慢的指针一次只移动一步。采用这种方法，在一个没有环的链表中，
当快的指针到达链表尾节点的时候慢的指针正好指向链表的中间节点。
